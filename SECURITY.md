# Security policy

## Reporting a vulnerability

If you find a security problem in Harmonica, report it privately to
<contact@harmonica.org.uk> rather than opening a public issue. Include what you
found, how to reproduce it, and what a fix might look like if you have one.

## Scope

By default the daemon binds to 127.0.0.1 and exposes nothing to other devices.
If your report concerns a network deployment, consider including the relevant
details of your setup.
