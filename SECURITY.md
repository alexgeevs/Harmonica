# Security policy

## Reporting a vulnerability

If you find a security problem in Harmonica, report it privately to
<contact@harmonica.org.uk> rather than opening a public issue. Include what you
found, how to reproduce it, and what a fix might look like if you have one. You
will normally get a reply within a few days, and credit in the release notes if
you want it.

## Scope worth knowing

Harmonica is a self-hosted daemon. By default it binds to 127.0.0.1 and exposes
nothing to other devices. Exposing it on a network is an explicit opt-in, so for
reports about a network deployment, please describe the configuration you used.

## Supported versions

Fixes land on the latest release only.
